# Anthropic timeline (Launch → March 17, 2026)

A verified, date-based timeline focused on major Anthropic company milestones, Claude model releases, and major product/platform feature launches (Claude app + API).

## Timeline

| Date (UTC) | What shipped | Category | Notes / what changed |
|---|---|---|---|
| 2021 | Anthropic founded | Company | Founded as an AI safety and research company (public benefit corporation). |
| Apr 29, 2022 | Series B announced | Company | Major early financing milestone to build “safe, reliable” AI systems and infrastructure. |
| Dec 15, 2022 | Constitutional AI paper (arXiv v1) | Research | Introduced “Constitutional AI” approach for training a harmless assistant via AI feedback. |
| Mar 14, 2023 | Claude + Claude Instant introduced | Model / Product | Claude announced; two versions offered: Claude and Claude Instant (plus broader availability beyond closed alpha). |
| Jul 11, 2023 | Claude 2 + claude.ai public beta | Model / Product | Claude 2 announced; public-facing claude.ai beta; API access noted. |
| Sep 7, 2023 | Claude Pro | Product | Paid plan introduced for claude.ai (initially US/UK). |
| Nov 21, 2023 | Claude 2.1 | Model | Claude 2.1 announced (200K context; improved reliability; new “tool use” beta noted). |
| Dec 19, 2023 | Beta Messages API + commercial/legal updates | Platform | Beta Messages API highlighted as a streamlined developer experience; expanded commercial terms/copyright indemnity changes announced. |
| Mar 4, 2024 | Claude 3 family (Haiku, Sonnet, Opus) | Model | Next-generation Claude 3 models announced. |
| May 1, 2024 | Claude Team plan + iOS app | Product | Team plan and iOS app announced. |
| May 14, 2024 | Claude available in Europe | Product | Claude availability expanded to Europe (including Pro/Team pricing in local currency). |
| Jun 21, 2024 | Claude 3.5 Sonnet | Model | First Claude 3.5-family release. |
| Aug 27, 2024 | Artifacts generally available | Product feature | Artifacts rolled out broadly across plans and mobile apps (after launching as a feature preview earlier in 2024). |
| Oct 22, 2024 | Upgraded Claude 3.5 Sonnet + Claude 3.5 Haiku + “computer use” (beta) | Model / Product feature | Upgraded 3.5 Sonnet + new 3.5 Haiku; “computer use” capability introduced in beta (agent-style desktop interaction). |
| Oct 24, 2024 | “Analysis tool” in Claude.ai | Product feature | Claude.ai feature allowing Claude to write/run JavaScript for analysis (later slated to be replaced by broader code execution capabilities). |
| Oct 31, 2024 | Claude desktop apps (Mac/Windows beta) | Product | Desktop apps announced (public beta). |
| Nov 25, 2024 | Model Context Protocol (MCP) | Platform | MCP open-sourced as a standard for connecting assistants to tools/data; local server support referenced for Claude desktop. |
| Feb 24, 2025 | Claude 3.7 Sonnet + Claude Code | Model / Product | Claude 3.7 Sonnet introduced as a “hybrid reasoning” model; Claude Code introduced alongside. |
| Mar 20, 2025 | Web search in Claude | Product feature | Claude web search introduced (later expanded to broader availability). |
| Apr 9, 2025 | Max plan | Product | Max plan introduced for expanded usage and priority access. |
| May 1, 2025 | Integrations (remote MCP servers) | Product feature | Integrations introduced to connect Claude with remote MCP servers (beyond local-only desktop MCP). |
| May 22, 2025 | Claude 4 (Opus 4 + Sonnet 4) | Model | Claude 4 announced; hybrid modes (near-instant + extended thinking) described; available via plans and API. |
| Sep 29, 2025 | Claude Sonnet 4.5 | Model | Sonnet 4.5 announced (positioned as a major step for coding/agents/computer use). |
| Oct 20, 2025 | Claude Code on the web | Product | Claude Code on the web introduced (browser-based workflow). |
| Nov 12, 2025 | Claude Code on the web expanded (update) | Product | Research-preview availability expanded for Team/Enterprise premium seats (in addition to Pro/Max). |
| Jan 12, 2026 | Cowork (research preview) | Product | “Cowork” introduced as Claude Code for broader work (research preview). |
| Jan 22, 2026 | Claude’s new constitution | Company / Product | Anthropic publishes a new “constitution” describing Claude’s intended values and behavior. |
| Feb 5, 2026 | Claude Opus 4.6 | Model | Opus 4.6 announced (flagship Opus upgrade). |
| Feb 12, 2026 | Claude Enterprise self-serve | Product | Claude Enterprise made purchasable self-serve (no sales conversation required). |
| Feb 17, 2026 | Claude Sonnet 4.6 | Model | Sonnet 4.6 announced (1M token context in beta noted; broad capability upgrades). |
| Feb 25, 2026 | Anthropic acquires Vercept | Company | Acquisition framed as advancing Claude’s computer-use capabilities. |
| Mar 9, 2026 | Code Review for Claude Code | Product feature | Code Review capability introduced for Claude Code. |
| Mar 12, 2026 | Interactive visuals in conversation (charts/diagrams/visualizations) | Product feature | Claude begins creating interactive visuals directly in chat. |
| Mar 13, 2026 | 1M context generally available for Opus 4.6 + Sonnet 4.6 | Platform | 1M context becomes generally available with standard pricing across the full window. |

## Notes on scope
- Anthropic (and the Claude platform) ships frequent incremental improvements. This timeline focuses on major named launches (models + headline product/platform features).
- If you want a “release-notes level” version (every update entry), the right source is the Claude blog “Product announcements” feed and the Claude developer docs release notes.
