# OpenAI timeline (Launch → March 17, 2026)

A verified, date-based timeline focused on major OpenAI company milestones, model releases, and major product/platform feature launches (ChatGPT + API).

## Timeline

| Date (UTC) | What shipped | Category | Notes / what changed |
|---|---|---|---|
| Dec 11, 2015 | OpenAI announced | Company | OpenAI publicly launches as a non-profit AI research company. |
| Jun 2018 | GPT-1 | Model | First GPT research release (“Improving Language Understanding by Generative Pre-Training”). |
| Feb 14, 2019 | GPT-2 (staged release strategy) | Model | “Better language models and their implications.” |
| Jun 11, 2020 | OpenAI API announced (early) | Platform | OpenAI describes the API program publicly. |
| May 2020 | GPT-3 | Model | “Language models are few-shot learners.” |
| Jan 5, 2021 | DALL·E (original) | Model | Early text-to-image milestone. |
| Sep 21, 2022 | Whisper | Model | Open-source speech recognition system. |
| Nov 30, 2022 | ChatGPT (research preview) | Product | ChatGPT launches publicly. |
| Mar 14, 2023 | GPT-4 | Model | GPT-4 announced/released for ChatGPT and API pathways. |
| Jul 2023 | Function calling + API updates | Platform | Function calling and other API updates released. |
| Nov 6, 2023 | GPT-4 Turbo + Assistants API + DALL·E 3 API + JSON mode/seed | Model + Platform | DevDay bundle: GPT-4 Turbo, Assistants API (incl. Retrieval + Code Interpreter), JSON mode, seed control, etc. |
| Nov 6, 2023 | GPTs (custom ChatGPTs) | Product | Custom versions of ChatGPT (“GPTs”) introduced. |
| Feb 15, 2024 | Sora (research) | Model (research) | Video generation described in OpenAI’s technical post. |
| Apr 24, 2024 | GPT-3.5 Turbo + Whisper APIs (positioning/availability updates) | Platform | API announcement for GPT-3.5 Turbo + Whisper; “ChatGPT API” naming clarified. |
| May 13, 2024 | GPT-4o (“omni”) | Model | Real-time multimodal model (text/vision/audio). |
| Jul 18, 2024 | GPT-4o mini | Model | Cost-efficient model in the 4o family. |
| Sep 12, 2024 | o1-preview + o1-mini | Model | Reasoning-focused o-series begins. |
| Dec 9, 2024 | Sora System Card | Safety/tech | Safety/system documentation for Sora-era model. |
| Feb 2, 2025 | Deep Research | Product feature | Deep research feature introduced in ChatGPT. |
| Feb 27, 2025 | GPT-4.5 (research preview) | Model | GPT-4.5 introduced. |
| Mar 11, 2025 | New tools for building agents | Platform | Agents-oriented tools announcement. |
| Mar 20, 2025 | Next-generation audio models in API | Platform | New STT/TTS/audio model line for the API. |
| Mar 25, 2025 | 4o Image Generation | Model capability | Image generation integrated with GPT-4o. |
| Apr 14, 2025 | GPT-4.1 / 4.1 mini / 4.1 nano (API) | Model | GPT-4.1 model family introduced for the API (large context support described). |
| Apr 16, 2025 | o3 + o4-mini | Model | Next o-series step; tool-using reasoning models. |
| May 16, 2025 | Codex (cloud software engineering agent) | Product | Codex introduced as an agent product (powered by codex-1). |
| Aug 5, 2025 | gpt-oss (open-weight models) | Model | gpt-oss-120b and gpt-oss-20b introduced. |
| Aug 7, 2025 | GPT-5 | Model | GPT-5 introduced. |
| Aug 28, 2025 | Realtime API updates (gpt-realtime) | Platform | Realtime API and related updates introduced. |
| Sep 30, 2025 | Sora 2 | Model | Sora 2 introduced. |
| Oct 6, 2025 | Apps in ChatGPT + Apps SDK (preview) | Product/Platform | Apps in ChatGPT and Apps SDK introduced. |
| Nov 12, 2025 | GPT-5.1 | Model | GPT-5.1 introduced. |
| Dec 11, 2025 | GPT-5.2 | Model | GPT-5.2 introduced. |
| Dec 16, 2025 | New ChatGPT Images | Product feature | New ChatGPT Images experience introduced. |
| Dec 18, 2025 | GPT-5.2-Codex | Model | GPT-5.2-Codex introduced. |
| Jan 16, 2026 | ChatGPT Go (worldwide) | Product | ChatGPT Go announced as available worldwide. |
| Mar 11, 2026 | GPT-5.1 retired in ChatGPT (migration) | Product/Platform | ChatGPT release notes indicate GPT-5.1 models removed from ChatGPT availability. |
| Mar 2026 (by Mar 17) | GPT-5.4 (Thinking + Pro) | Model | GPT-5.4 introduced (ChatGPT includes GPT-5.4 Thinking; Pro variant referenced). |

## Notes on scope
- OpenAI ships many incremental changes (especially in ChatGPT). This timeline focuses on major named launches.
- For a fully exhaustive “every feature release” chronology, the authoritative source is the official ChatGPT release notes.
